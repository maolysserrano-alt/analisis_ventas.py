#Análisis de Ventas - Python Core

Este proyecto aplica el uso de **listas, tuplas y diccionarios** para procesar datos de ventas y obtener métricas clave.

## 🛠️ Conceptos Aplicados
* **Diccionarios Anidados:** Para el resumen final de productos.
* **Tuplas:** Utilizadas para almacenar pares de datos inmutables (suma de precios y cantidad).
* **Bucles Iterativos:** Para recorrer la lista de ventas original.
